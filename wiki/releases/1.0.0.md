# LLM Workbench v1.0.0 Release Notes

LLM Workbench v1.0.0 delivers a production-ready AI assistant featuring a dual LLM backend (local llama.cpp and cloud Claude) and a modular plugin architecture. The release introduces a comprehensive `source_*` tool family for self-aware code management, including safe editing, git operations, and project switching. It implements the TaskMaster pipeline for context isolation to optimize memory usage on local models, alongside robust web research capabilities using Tavily, Keiro, and DuckDuckGo with stealth and browser fetch modes. Key integrations include full E*TRADE brokerage access for trading and portfolio analysis, LiteAPI for hotel booking, and a visualization system supporting SVG, Chart.js, and custom HTML applets.

## Key topics
- Source Tools — Self-modifying codebase management with safety tiers and git integration
- TaskMaster — Context-isolated pipelines to prevent memory bloat in local LLMs
- Web Research — Multi-engine search with stealth scraping and full browser execution
- E*TRADE API — Real-time quotes, options Greeks, and transaction history export
- Travel Booking — LiteAPI integration for hotel search, rates, and fraud-checked booking
- Dual Backend — Hybrid routing between local privacy-focused models and cloud reasoning
- Visualization — Inline iframe applets for charts, diagrams, and custom HTML layouts
- Plugin System — Hot-reloadable modules with dependency graph validation
